# 📖 OOP school library <a name="about-project"></a>
_**OOP school library**_ can be used to manage library information (users, books, borrows).

<!-- Features -->

### Key Features <a name="key-features"></a>

- Create users: Students and Teachers
- Create books
- Create borrows
- List all books
- List all users
- List all borrows
- List all borrows for a given user
- List all borrows for a given book

## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project you need:

- [Ruby](https://www.ruby-lang.org/en/) installed and running
### Setup

Clone this repository to your desired folder:

```
  git clone https://github.com/Camilovelag/OOP-school-library.git
  cd OOP-school-library
```
